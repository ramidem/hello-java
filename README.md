# Hello, Java!

[Learning Java](https://www.linkedin.com/learning/paths/become-a-java-programmer) for 30 mins a day! just because.

## Running the program

1. Compile it first.
    ```sh
    javac src.Main.java
    ```

2. Run the program to output `Hello, World!`.
    ```sh
    java src.Main
    ```
